# libisor.com 프론트

## created by react.js

* ts적용
* react-bootstrap

초안의 개괄
```
React 사용
데스크톱, 모바일 하나만 개발 하거나 복잡하게 반응형으로 만들 필요 없이
콘텐츠 틀만 데스크톱의 경우 대충 스크린 50% 정도(적당히 판단)
모바일의 경우 스크린 100% 사용하게 하고 나머지는 %, vh 같은거 써서 데스크탑, 모바일에서 비슷하게 보이게 개발하면 됨.
PPT 예상 화면 참고,

큰 틀에서 상단바와 콘텐츠 영역으로 나뉨.
서버와 통합 전 각 화면 틀 
```
